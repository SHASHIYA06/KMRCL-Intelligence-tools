# KMRCL Metro Intelligence

A React-based AI-powered document analysis and intelligence platform for Kolkata Metro Rail Corporation Limited (KMRCL). The application provides metro engineers with intelligent document search, analysis, and AI-assisted workflows.

## Core Features

- **Document Analysis**: PDF, DOCX, XLSX file processing with AI-powered insights
- **Intelligence Hub**: Multi-mode AI chat (Engineering, General, Email, Letter drafting)
- **Drive Browser**: Google Drive integration for document management
- **Voice Agent**: Voice-controlled document selection and analysis
- **Role-based Access**: Admin and User roles with granular permissions
- **Circuit Analysis**: Specialized engineering circuit component extraction

## Target Users

- Metro engineers and technical staff
- Administrative personnel
- Document analysts
- Project managers

The platform integrates with Google's Gemini AI for advanced document understanding and uses a modern React frontend with TypeScript for type safety.
